# Tonalität und Stigma Styleguide

Dieses Dokument soll helfen, neue Inhalte der BipolarSite sprachlich warm, fachlich klar und möglichst nicht-stigmatisierend zu halten. Es ersetzt keine redaktionelle Prüfung, gibt aber eine kurze gemeinsame Linie vor.

## Leitlinien

1. Menschen nie auf Diagnose, Risiko oder Krisenverhalten reduzieren.
2. Krisen klar benennen, ohne Personen als "ausser Kontrolle" oder "zerstörerisch" zu rahmen.
3. Angehörige entlasten, ohne betroffene Menschen abzuwerten.
4. In Lernmodulen eher einordnend als dramatisierend formulieren.
5. In Notfalltexten darf die Sprache direkter und kürzer sein, wenn sie handlungsleitend bleiben muss.
6. Wiederholungen von "die erkrankte Person" vermeiden, wenn eine menschlichere Variation gleich präzise ist.
7. Nicht mit "richtig/falsch" arbeiten, wenn "eher hilfreich/eher riskant" denselben Lerneffekt ohne Beschämung erzielt.
8. Symptome nicht moralisch deuten: lieber "Symptom", "Zuspitzung", "eingeschränktes Urteilsvermögen" als "Willensschwäche" oder "Sturheit".
9. Paarbeispiele dürfen bleiben, aber Eltern, Geschwister und erwachsene Kinder sollen sprachlich mitgemeint sein.
10. Zitate transparent kennzeichnen: anonymisiert, typisierte Erfahrungsstimme oder Fallvignette.

## Lieber so / vermeiden

| Vermeiden | Lieber so |
| --- | --- |
| "die Bipolaren" | "Menschen mit bipolarer Störung" |
| "psychisch Kranker" | "psychisch erkrankte Menschen" |
| "Manie ausser Kontrolle" | "manische Krise mit hohem Risiko" |
| "massive Entgleisung" | "schwere Krise" oder "akute Zuspitzung" |
| "finanzielle Zerstörung" | "finanzielle Schäden und existenzielle Folgen" |
| "wenn es kippt" | "wenn sich die Lage verschlechtert" oder "wenn sich die Lage zuspitzt" |
| "falsche Antwort" | "eher riskante Antwort" |
| "richtige Antwort" | "eher hilfreiche Antwort" |
| ständige Wiederholung von "die erkrankte Person" | "die betroffene Person", "Ihr Angehöriger / Ihre Angehörige", "die Person", "Ihr Gegenüber" |
| "Patient" ausserhalb von Behandlungskontexten | "betroffene Person" oder "Mensch mit bipolarer Störung" |

## Projektnahe Beispiele

- Notfallseite: klar und handlungsleitend bleiben. "Gewalt oder Bedrohung" ist sinnvoll; "Zerstörung" nur verwenden, wenn konkret Sachbeschädigung gemeint ist.
- Module: eher "belastende Muster", "akute Zuspitzung" oder "schwer erreichbar" als dramatische Totalformeln.
- Tools: Rückmeldungen sollen lernorientiert klingen. Nicht "Sie hatten X von Y richtig", sondern eher "Sie haben häufig deeskalierende Antworten gewählt".
- Selbsttests: immer als Momentaufnahme rahmen. Keine Diagnose, keine Bewertung der Person.
- Erfahrungsstimmen: lieber "typisierte Erfahrungsstimme, anonymisiert" oder "Fallvignette, aus typischen Angehörigenerfahrungen verdichtet" als eine unklare Echtheitsanmutung.

## Merksatz

Notfalltexte dürfen klarer und direkter sein. Lernmodule und Tools sollen stärker entlastend, differenzierend und menschenbezogen formulieren.
